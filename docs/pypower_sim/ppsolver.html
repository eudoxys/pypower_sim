pypower_sim.ppsolver

PyPOWER solver

This model implements the PyPOWER steady-state powerflow (PF), optimal powerflow (OPF), and quasi-steady time series (QSTS) solvers.

Description

The following solvers are available:

  1. Powerflow (PF)

    pypower_sim.ppsolver.PPSolver.solve_pf: solves steady-state powerflow (PF) problem

  2. Optimal Powerflow (OPF)

    pypower_sim.ppsolver.PPSolver.solve_opf: solve optimal powerflow problem (DC or AC OPF).

  3. Optimal Sizing and Placement (OSP)

    pypower_sim.ppsolver.PPSolver.solve_osp: solve optimal sizing/placement problem (OSP).

  4. Time-series Simulation (QSTS)

    pypower_sim.ppsolver.PPSolver.run_timeseries: solver quasi-steady time-series (QSTS) problems.

Example

This example loads the WECC 240 case and solve an hourly timeseries simulation for August 2020.

from wecc240 import wecc240
model = PPModel(case=wecc240)

solver = PPSolver(model)
solver.run_timeseries(
    start="2020-08-01 00:00:00+07:00",
    end="2020-09-01 00:00:00+07:00",
    freq="1h",
    )

135    def solve_osp(self,
136        options:dict[str:str|int|float|dict]|None=None,
137        update:str='success',
138        with_result:bool=False,
139        costs:dict[str,float]|None=None,
140        generators:dict[str,float]={
141            "gen":{
142                "GEN_STATUS": 1,
143                "MBASE": 100,
144                },
145            "gencost":{
146                "MODEL": 2, # polynomial function
147                "NCOST": 2, # linear cost
148                "COST0": 50, # operating cost ($/MWh)
149                "COST1": 0, # fixed cost ($/h)
150                },
151            "roundup":-1,
152            },
153        capacitors:dict[str,float]={
154            "gen":{
155                "GEN_STATUS": 1,
156                "MBASE": 100,
157                },
158            "gencost":{
159                "MODEL": 2, # polynomial function
160                "NCOST": 2, # linear cost
161                "COST0": 0, # operating cost ($/MWh)
162                "COST1": 0, # fixed cost ($/h)
163                },
164            "roundup":0,
165            },
166        condensers:dict[str,float]={
167            "gen":{
168                "GEN_STATUS": 1,
169                "MBASE": 100,
170                },
171            "gencost":{
172                "MODEL": 2, # polynomial function
173                "NCOST": 2, # linear cost
174                "COST0": 0, # operating cost ($/MWh)
175                "COST1": 0, # fixed cost ($/h)
176                },
177            "roundup":0,
178            },
179        ):
180        """Solve the optimal sizing placement problem
181
182        # Arguments
183
184        - `options`: specify problem options to enable/disable
185          (see `pypower_sim.runosp.OspConfig`)
186
187        - `update`: when to update of model case data (must be in `
188          {'always', 'success', 'failure', 'never'}`)
189    
190        - `with_result`: include result in return value
191
192        - `costs`: specify capacity expansion costs
193
194        - `generators`: default `gen`, `gencost`, and `roundup` values for new
195          generators
196
197        - `capacitors`: default `gen`, `gencost`, and `roundup` values for new
198          generators
199
200        - `condensers`: default `gen`, `gencost`, and `roundup` values for new
201          generators
202
203        # Returns
204
205        - `bool`: `True` on success, `False` on failure
206
207        - `dict`: result (if `with_result` is `True`)
208
209        # Description
210
211        The following construction `costs` may specified:
212
213        1. `capacitor`: specifies the capacitor addition cost in $/MVAr
214        (default `100`)
215
216        2. `condenser`: specifies the condenser addition cost in $/MVAr
217        (default `1000`)
218
219        3. `generation`: specifies the generation addition cost in $/MVA
220        (default `1000`)
221
222        4. `curtailment`: specifies the load curtailment cost in $/MW
223        (default `10000`)
224
225        Note that in general the costs should increase in the order presented
226        above, i.e., capacitors are the least costly and load curtailment is
227        the most costly.
228
229        ## Model Updates
230
231        The model is updated as follows:
232
233        1. Generators are added to non-PQ busses. Only the real power capacity
234        `PMAX` is set initially. 
235
236        2. Dynamic capacitors and condensers are added to non-PQ busses using
237        generators with zero real power `PMAX` by setting `QMAX` and `QMIN`
238        respectively.
239
240        3. Static capacitors and condensers are added to PQ busses by adding
241        the indicated reactive power to `BS`.
242
243        4. Generators cannot be added to non-PQ busses. To enable additional
244        generation to a bus, the `BUS_TYPE` must be set to `PV` (2) or `REF`
245        (3).
246
247        # Caveat
248
249        The optimal sizing/placement solver is experimental.
250        """
251        # warnings.warn("solver_osp is not implemented yet")
252        result = runosp(self.model,config=options)
253        status = result["status"]
254        success = status == 1
255        if ( success and update in ["always","success"] ) \
256                or ( not success and update in ["always","failure"] ):
257
258            # update gen and gencost data
259            bus = self.model.get_data("bus")
260            gen = self.model.get_data("gen")
261            gencost = self.model.get_data("gencost")
262            pqbus_ndx = bus[bus.BUS_TYPE == idx_bus.PQ].index.values.astype(int)
263
264            # add new generators to gen busses
265            if sum(result["generators"]) > 0:
266                gen_bus, pmax = np.array([(int(x),y) for x,y in enumerate(result["generators"]) if round(y,3) > 0 and int(x) not in pqbus_ndx]).T
267                if "roundup" in generators:
268                    pmax = np.ceil(pmax*10**generators["roundup"])/10**generators["roundup"]
269                newgen = {
270                    "GEN_BUS":bus.iloc[gen_bus].BUS_I,
271                    "PMAX":pmax,
272                    "VG": np.abs(result["voltages"][gen_bus.astype(int)]).round(3)
273                    }
274                newgen.update({x:[y]*(len(gen_bus)) for x,y in generators["gen"].items()})
275                newgencost = {x:[y]*(len(gen_bus)) for x,y in generators["gencost"].items()}
276            else:
277                newgen = {x:[] for x in gen.columns}
278                newgencost = {x:[] for x in gencost.columns}
279
280            # add active capacitors to gen busses
281            if sum(result["capacitors"]) > 0:
282                cap_bus, qmax = np.array([(x,y) for x,y in enumerate(result["capacitors"]) if round(y,3) > 0 and int(x) not in pqbus_ndx]).T
283                if "roundup" in capacitors:
284                    qmax = np.ceil(qmax*10**capacitors["roundup"])/10**capacitors["roundup"]
285                newcap = {
286                    "GEN_BUS": bus.iloc[cap_bus].BUS_I,
287                    "QMAX":qmax,
288                    "VG": np.abs(result["voltages"][cap_bus.astype(int)]).round(3)
289                    }
290                newcap.update({x:[y]*(len(cap_bus)) for x,y in capacitors["gen"].items()})
291                newcapcost = {x:[y]*(len(cap_bus)) for x,y in capacitors["gencost"].items()}
292            else:
293                newcap = {x:[] for x in gen.columns}
294                newcapcost = {x:[] for x in gencost.columns}
295
296            # add active condensers to gen busses
297            if sum(result["condensers"]) > 0:
298                con_bus, qmin = np.array([(x,y) for x,y in enumerate(-result["condensers"]) if round(y,3) > 0 and int(x) not in pqbus_ndx]).T
299                if "roundup" in condensers:
300                    qmin = np.ceil(qmin*10**condensers["roundup"])/10**condensers["roundup"]
301                newcon = {
302                    "GEN_BUS": bus.iloc[con_bus].BUS_I,
303                    "QMIN": qmin,
304                    "VG": np.abs(result["voltages"][cons_bus.astype(int)]).round(3)
305                    }.update(condensers["gen"])
306                newcon.update({x:[y]*(len(con_bus)) for x,y in condensers["gen"].items()})
307                newconcost = {x:[y]*(len(con_bus)) for x,y in condensers["gencost"].items()}
308            else:
309                newcon = {x:[] for x in gen.columns}
310                newconcost = {x:[] for x in gencost.columns}
311
312            # compile new gen array
313            gen = pd.concat([gen,
314                pd.DataFrame(newgen),
315                pd.DataFrame(newcap),
316                pd.DataFrame(newcon),
317                ]).fillna(0).reset_index(drop=True)
318            self.model.case["gen"] = gen.values
319
320            # compile new gencost array
321            if len(gencost) > 0:
322                gencost = pd.concat([gencost,
323                    pd.DataFrame(newgencost),
324                    pd.DataFrame(newcapcost),
325                    pd.DataFrame(newconcost),
326                    ]).fillna(0).reset_index(drop=True)
327                self.model.case["gencost"] = gencost.values
328
329            # add passive capacitors to PQ busses
330            if sum(result["capacitors"]) > 0:
331                cap_bus, qmax = np.array([(x,y) for x,y in enumerate(result["capacitors"]) if round(y,3) > 0 and int(x) in pqbus_ndx]).T
332                if "roundup" in capacitors:
333                    bs = np.ceil(qmax*10**capacitors["roundup"])/10**capacitors["roundup"]
334                newcap = {
335                    "BUS": self.model._bus_i(bus.iloc[cap_bus].BUS_I.values.tolist()),
336                    "BS":bs,
337                    }
338                self.model.case["bus"][newcap["BUS"],idx_bus.BS] += newcap["BS"]
339
340            # add passive condensers to PQ busses
341            if sum(result["condensers"]) > 0:
342                con_bus, qmin = np.array([(x,y) for x,y in enumerate(-result["condensers"]) if round(y,3) > 0 and int(x) in pqbus_ndx]).T
343                if "roundup" in condensers:
344                    bs = -np.ceil(qmin*10**condensers["roundup"])/10**condensers["roundup"]
345                newcon = {
346                    "BUS":self.model._bus_i(bus.iloc[con_bus].BUS_I.values.tolist()),
347                    "BS": bs,
348                    }
349                self.model.case["bus"][newcap["BUS"],idx_bus.BS] += newcap["BS"]
350
351        if with_result:
352            return status,result
353        else:
354            return status

356    def update_inputs(self,t:dt.datetime) -> int:
357        """Synchronize inputs with the current date/time
358
359        # Arguments
360
361        - `t`: the current date/time
362        """
363        # update inputs
364        errors = 0
365        for name,spec in self.model.inputs.items():
366            data = spec["data"]
367            name,column = name
368            column_number = self.model.get_header(name).index(column)
369            mapping = spec["mapping"]["index"]
370            scales = spec["mapping"]["scale"]
371            try:
372                target = self.model.case[name]
373                target[mapping,column_number] = data.loc[t] * scales
374            except KeyError as exception:
375                warnings.warn(f"input({name=},{column=}) {exception=}")
376                errors += 1
377        return errors

379    def update_outputs(self,
380        t:dt.datetime,
381        ts_format:str="%Y-%m-%d %H:%M:%S %Z"
382        ) -> int:
383        """Synchronize outputs to the current date/time
384
385        # Arguments
386
387        - `t`: the current date/time
388
389        - `ts_format`: timestamp format
390        """
391        ts = t.strftime(ts_format)
392
393        # output
394        for file,spec in self.model.outputs.items():
395            mapping = spec["mapping"]
396            scale = mapping["scale"]
397            offset = mapping["offset"]
398            data = [f"{{0:{spec['format']}}}".format(x)
399                for x in self.model.get_data(
400                    spec["name"]).loc[mapping["rows"],spec["column"]]*scale + offset]
401            print(ts,*data,sep=",",file=spec["fh"],flush=True)
402
403        # recorders
404        errors = 0
405        for file,recorder in self.model.recorders.items():
406            values = [ts]
407            for _,spec in recorder["targets"].items():
408                value = self.model.case
409                for level in spec["source"]:
410                    if not level in value:
411                        warnings.warn(f"recorder '{file}' -- '{level}' not found")
412                        errors += 1
413                        break
414                    value = value[level]
415                if not isinstance(value,(int,float,bool,str,type(None))):
416                    value = float('nan')
417                elif isinstance(value,(int,float)):
418                    value = eval(spec["transform"])(value)
419                values.append(f"{{0:{spec['format']}}}".format(value))
420            print(*values,sep=",",file=recorder["fh"],flush=True)
421        return errors

423    def run_timeseries(self,*args,
424        # pylint: disable=too-many-arguments,too-many-locals
425        progress:Callable=None,
426        call_on_fail:Callable=None,
427        stop_on_fail:bool=True,
428        stop_test:Callable=None,
429        use_acopf:bool=True,
430        **kwargs) -> str|list[str]|None:
431        """Run a timeseries simulation
432
433        Time-series solutions perform the following sequence of operations at each
434        timestep.
435
436        ```mermaid
437        sequenceDiagram
438
439            autonumber
440            note over inputs, players: Read data
441
442            note over outputs, recorders: Open files
443
444            loop from start to end by timestep
445
446                inputs ->>model:Input data
447
448                players ->>model:Player data
449
450                model ->>+solvers:Initial/last result
451                note over solvers: Call runopf()
452                solvers ->>-model:OPF result
453
454                model ->>+solvers:OPF result
455                note over solvers: Call runpf()
456                solvers ->>-model:PF result
457
458                note over model: Save errors
459
460                model ->>+outputs:Output data
461
462                model ->>+recorders:Recorder data
463            
464            end
465
466            note over model: Generate profile
467        ```
468
469        # Arguments
470
471        - `*args`: See `pandas.date_range(*args)`
472
473        - `progress`: set a progress callback function
474
475        - `call_on_fail`: set a call-on-fail function
476
477        - `stop_on_fail`: enable stop-on-fail condition
478
479        - `stop_test`: set a stop test call back function
480
481        - `use_acopf`: enable use of AC OPF instead of DC OPF
482
483        - `**kwargs`: See `pandas.date_range(**kwargs)`
484        
485        # Returns
486
487        - `None`: No errors to report
488
489        - `str`: Error message (when stop_on_fail is True)
490
491        - `list[str]`: Error messages (when stop_on_fail is False)
492        """
493
494        assert progress is None or callable(progress), \
495            "progress must be callable or None"
496        assert call_on_fail is None or callable(call_on_fail), \
497            "call_on_fail must be callable or None"
498        assert stop_test is None or callable(stop_test), \
499            "stop_test must be callable or None"
500        assert isinstance(stop_on_fail,bool), \
501            "stop_on_fail must be boolean"
502        assert isinstance(use_acopf,bool), \
503            "use_acopf must be boolean"
504
505        self.model.errors = [] # collect errors, if any
506        tic0 = time()
507
508        # process time specified range
509        trange = pd.date_range(*args,**kwargs)
510        niters = 0
511        topf = 0.0
512        tpf = 0.0
513
514        # start recorders
515        for file,recorder in self.model.recorders.items():
516            recorder["fh"] = open(file,"w",encoding="utf-8")
517            columns = ["timestamp"] + list(recorder["targets"].keys())
518            print(*columns,sep=",",file=recorder["fh"],flush=True)
519
520        # start outputs
521        for file,output in self.model.outputs.items():
522            output["fh"] = open(file,"w",encoding="utf-8")
523            columns = ["timestamp"] + output["mapping"]["columns"]
524            print(*columns,sep=",",file=output["fh"],flush=True)
525
526        for t in (x.tz_convert("UTC") for x in trange):
527
528            # setup time and progress/stop callback
529            ts = t.strftime("%Y-%m-%d %H:%M:%S %Z")
530            if callable(progress) and progress(f"""{ts} ({len(self.model.errors)
531                    if self.model.errors else 'no'} errors)"""):
532                return None
533
534            # update inputs
535            if self.update_inputs(t) > 0:
536                if call_on_fail:
537                    call_on_fail("Input error")
538                if stop_on_fail:
539                    break
540
541            # solve OPF and check result
542            status,result = self.solve_opf(use_acopf,with_result=True)
543            if status is not True:
544                failed = f"OPF failed at {ts}"
545                self.model.errors.append(failed)
546                if call_on_fail:
547                    call_on_fail(failed)
548                if stop_on_fail:
549                    break
550            topf += result["et"]
551
552            # solver powerflow and check result
553            status,result = self.solve_pf(with_result=True)
554            if status is not True:
555                failed = f"PF failed at {ts}"
556                self.model.errors.append(failed)
557                if call_on_fail:
558                    call_on_fail(failed)
559                if stop_on_fail:
560                    break
561            tpf += result["et"]
562
563            # process outputs
564            if self.update_outputs(t) > 0:
565                if call_on_fail:
566                    call_on_fail("Output error")
567                if stop_on_fail:
568                    break
569
570            # check stop condition
571            niters += 1
572            if stop_test and stop_test(t):
573                self.model.errors.append(f"Stopped at {t=}")
574                break
575
576        ttot = time() - tic0
577        self.model.profile = {
578            "Expected iterations": len(trange),
579            "Completed iterations": niters,
580            "Total OPF time (s)": round(topf,4),
581            "Fraction OPF time (s/s)": round(topf/ttot,2) if ttot > 0 else 0,
582            "Total PF time (s)": round(tpf,4),
583            "Fraction PF time (s/s)": round(tpf/ttot,2) if ttot > 0 else 0,
584            "Total run time (s)": round(ttot,4),
585            "Iteration time (s/iter)": round(ttot/niters,4) if niters > 0 else "N/A",
586        }
587
588        return self.model.errors if self.model.errors else None
